# -MFC-
采用MFC框架，使用Visual Studio 2019进行项目开发。
-------------------------------
功能
服务器：
服务器的启动监听和关闭；
	服务器的主要功能是向客户端发送文件和发送消息；
客户端：
	客户端可以连接服务器
	客户端可以向服务器发送消息或发送文件。
